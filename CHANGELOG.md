# Changelog

All notable changes to MobileAgentOS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-XX

### Added

- Initial release
- @mobileagentos/core package
- @mobileagentos/react-native package
- @mobileagentos/benchmark package
- NOVA Demo reference app
- Mobile AI Readiness Framework
- Documentation

### Core Runtime

- AgentRuntime with full pipeline
- ContextEngine for multi-dimensional context
- MemoryEngine with InMemoryStore
- PrivacyGuard with PII redaction
- ActionRouter with confirmation flows
- FeedbackLoop for signal capture
- ReadinessScore with 7-dimension framework

### React Native Adapter

- MobileAgentProvider
- useAgent hook
- useAgentMemory hook
- useAgentActions hook
- useReadinessScore hook
- Basic UI components

### Benchmark Suite

- runReadinessBenchmark
- createReadinessReport
- formatReadinessScore
