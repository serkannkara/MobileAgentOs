# Contributing to MobileAgentOS

Thank you for your interest in contributing to MobileAgentOS!

## Philosophy

MobileAgentOS is built for mobile product builders shipping AI-native experiences.

**Not another chatbot starter** — a runtime, benchmark, and reference architecture for AI-native mobile products.

## How to Contribute

### 1. Issues

- Report bugs
- Suggest features
- Ask questions
- Share use cases

### 2. Pull Requests

- Fork the repository
- Create a feature branch
- Make your changes
- Add tests where applicable
- Submit a PR with clear description

### 3. Documentation

- Improve existing docs
- Add examples
- Fix typos
- Share patterns

## Development Setup

```bash
git clone https://github.com/mobileagentos/mobileagentos.git
cd mobileagentos
pnpm install
pnpm build
pnpm test
```

## Code Quality

- TypeScript strict mode
- Clear exported interfaces
- Readable names
- No `any` unless absolutely necessary
- Tests for core functionality

## Areas We Need Help

- Platform adapters (Compose, Swift)
- Real-world reference apps
- Benchmarking improvements
- Documentation improvements
- Privacy patterns
- Action safety patterns

## Community

MobileAgentOS is built by mobile product builders for mobile product builders.

Join us in building the open-source foundation for AI-native mobile apps.
