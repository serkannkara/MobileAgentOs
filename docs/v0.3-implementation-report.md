// ✅ **Privacy Guard** — Regex-based PII detection (6 types)

// ## Known Limitations
// 
// ### v0.3 Scope
// 
// As planned, v0.3 does NOT include:
// 
// ❌ Real LLM provider integration
// ❌ CoreData/SwiftData persistence
// ❌ Streaming responses
// ❌ Location/calendar/motion context
// ❌ Real iOS action permissions
// ❌ Keychain secure storage
// ❌ CloudKit sync
// ❌ Background tasks
// 
// These are intentionally deferred to v0.4+.
// 
// ### Current State
// 
// - **InMemoryStore** — Thread-safe with actors, but resets on app restart
// - **Deterministic responses** — "Processed: {input}" format
// - **Mock actions** — Safe demo actions with simulated execution
// - **Basic context** — Platform, locale, timezone, app version only
// - **Regex PII detection** — Pattern-based, not ML-based
